###############################
Supported Storage
###############################
GenomicsDB has support for several filesystems. 

Local / Cluster
*******************************
* *POSIX*: Tested on Centos 6/7 and Ubuntu 20.04 

Cloud
*******************************
* *AWS*: Support for S3 and EMRFS
* *GCP*: Support for GCP
* *Azure*: Support for Azure Blob storage and Azure Data Lake Storage Gen2
