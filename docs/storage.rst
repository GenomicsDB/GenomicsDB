###############################
Supported Storage
###############################
GenomicsDB has support for several filesystems. 

Local / Cluster
*******************************
* *POSIX*: Tested on Centos 6 + 7 

* *HDFS*: HDFS support

Cloud
*******************************
* *AWS*: Support for S3 and EMRFS
* *GCP*: Support for G3
* *Azure*: Support for Azure Blob storage
